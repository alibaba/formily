# @uform/react-schema-renderer
> Schema渲染引擎，该包主要依赖了@uform/react，它的职责很简单，核心就做了两件事情：
>
> - 解析Form Schema协议，递归渲染
> - 管理自定义组件

### 安装

```bash
npm install --save @uform/react-schema-renderer
```
