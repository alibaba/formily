# setValidationLanguage

## 介绍

设置校验错误提示的国际化语言

## 类型描述

```typescript
type setValidationLanguage = (lang: string) => void
```

## 依赖

```javascript
import {setValidationLanguage} from '@uform/core'
```

## 用例

```javascript
import {setValidationLanguage} from '@uform/core'

setValidationLanguage("en")
```
