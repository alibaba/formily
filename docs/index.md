## Install dependencies

```bash

$ npm install --save antd moment

```

or

```bash

$ npm install --save @alifd/next moment

```

## Install Formily

```bash
$ npm install --save @formily/core @formily/react @formily/antd

```

or

```bash
$ npm install --save @formily/core @formily/react @formily/next

```
